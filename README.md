# 🏠 Family & Personal Serverless Web Apps

这是一个基于 **HTML5 + JavaScript (Vanilla/Vue3)** 和 **LeanCloud Serverless** 架构的轻量级个人/家庭管理工具集。

本项目包含三个独立的单页应用 (SPA)，**无需购买服务器**，直接在浏览器运行，数据存储于您私人的 LeanCloud 云端，安全且完全由您掌控。

---

## 📂 应用列表

1. **💰 家庭资金管理系统** (expense-tracker.html)
    
    - 多维度收支记录、预算控制、可视化图表分析。
        
2. **📦 商品保质期管理系统** (shelf-life-manager.html)
    
    - 家庭物品库存管理、保质期监控、临期/过期状态自动提醒。
        
3. **📚 云端错题本** (error-notebook.html)
    
    - 支持图文混排的错题整理、遮挡复习模式及知识点关联。
        

---

## ⚙️ 数据库配置与自定义 (重要)

本项目使用 LeanCloud 作为后端数据库。代码中默认定义了一些表名（Class Name），**您可以直接使用默认名，也可以根据喜好在代码中查找并替换为自己的表名**。

### 1. 数据表对照表

请在 LeanCloud 控制台预先了解或创建以下 Class，或者直接运行代码让其自动创建：

|   |   |   |
|---|---|---|
|应用名称|代码中默认表名 (Class)|建议操作|
|**资金管理**|rourou_YellowChamberPot|建议在代码中全局搜索并替换为您喜欢的名称（如 FamilyExpense）|
|**商品管理**|rourou_BananaNest|建议在代码中全局搜索并替换为您喜欢的名称（如 ShelfLifeItem）|
|**错题本**|Question / Knowledge / Category|通用名称，可保留或修改|
|**用户系统**|_User|LeanCloud 内置表，**不可修改**|

### 2. 关于“家庭组”与数据隔离

为了支持多家庭使用或数据隔离，各应用的逻辑如下：

- **📦 商品保质期管理**：
    
    - **交互方式**：在登录界面或配置界面，有一个 **“家庭名称”** 输入框。
        
    - **逻辑**：只要多个家庭成员输入**相同的 AppID/Key** 和 **相同的家庭名称**，即可共享同一份数据，同时系统会记录具体的操作人（记录人）。
        
    - **无需改代码**：直接在界面操作即可加入现有家庭。
        
- **💰 资金管理系统**：
    
    - **交互方式**：目前家庭标识定义在代码变量中。
        
    - **逻辑**：请在代码头部找到 const MY_FAMILY_ID = "big banana house";。
        
    - **需要改代码**：如果您希望自定义家庭标识，请**修改此变量的值**。所有使用该代码文件的家庭成员，需保持此变量一致才能同步账本。

## 🚀 快速开始

### 1. 获取后端凭证 (LeanCloud)

1. 注册并登录 [LeanCloud 控制台](https://www.google.com/url?sa=E&q=https%3A%2F%2Fconsole.leancloud.cn%2F) (或国际版)。
    
2. 创建一个新应用（例如命名为 "MyFamilyApps"）。
    
3. 进入 **设置 > 应用凭证**，获取 AppID 和 AppKey。
    
4. 进入 **设置 > 域名绑定**，绑定 API 域名（国际版可跳过）。
    

### 2. 配置与运行

1. 下载本项目的所有 .html 文件。
    
2. **自定义修改**（可选）：如前文所述，使用文本编辑器打开 html 文件，查找并替换默认的表名（如 rourou_BananaNest），修改资金管理系统中的 MY_FAMILY_ID。
    
3. **运行**：直接用浏览器（推荐 Chrome/Edge）打开 html 文件。
    
4. **初始化**：首次运行时弹出配置框，填入您的 AppID、AppKey 和 ServerURL。配置信息将安全地存储在您的本地浏览器缓存中。

## 🛠️ 功能特性详解

### 1. 💰 家庭资金管理

- **多维度报表**：包含柱状图（预算监控）、饼图（消费占比）。
    
- **智能分析**：内置 17 种分析模型（如“周末vs工作日消费对比”、“单笔最高消费”等），点击刷新随机展示。
    
- **预算管理**：直观的进度条显示本月剩余预算及百分比。
    

### 2. 📦 商品保质期管理

- **状态自动计算**：根据生产日期+保质期，自动标记 **正常/临期/过期**。
    
- **数据认领**：特有的“一键认领”功能，可将历史无主数据快速同步到当前登录账号和家庭名下。
    
- **协作记录**：多人协作时，列表清晰展示每一条物品是由哪位家庭成员录入的。
    

### 3. 📚 云端错题本

- **图文支持**：支持 Markdown 语法，**支持直接粘贴截图**（自动上传至 LeanCloud 文件存储）。
    
- **刷题模式**：
    
    - **遮挡模式**：默认隐藏答案和解析，点击查看。
        
    - **斩杀机制**：标记为“已掌握”的题目将不再出现在刷题队列中。
